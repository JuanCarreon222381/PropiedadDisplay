Propiedad Display

Ideas Principales.

  • Controla el tipo de caja y el comportamiento de los elementos
  • Soporte para layouts avanzados: Flexbox y Grid
  • Se pueden ocultar elementos con display:none

Apuntes de Clase.

display: block;

Hace que el elemento se comporte como un bloque. Ocupa todo el ancho disponible en su contenedor y siempre comienza en una nueva línea.

display: inline;

Hace que el elemento se comporte como un elemento en línea. No ocupa el ancho completo y no causa saltos de línea, se coloca junto a otros elementos.

display: inline-block;

Combina las características de los elementos inline y block. Se comporta como un elemento en línea, pero puedes definir su width y height.

display: none;

Hace que el elemento no se muestre en la página en absoluto. El elemento se elimina del flujo del documento, por lo que no ocupa espacio en la página.

display: flex;

Aplica el modelo de diseño flexbox a un contenedor. Los elementos hijos de un contenedor con display: flex; se alinean en una fila o columna, dependiendo de la dirección del contenedor.

display: grid;

Aplica el modelo de diseño grid a un contenedor. Permite dividir el espacio del contenedor en una cuadrícula, en la que los elementos hijos se organizan en filas y columnas.
